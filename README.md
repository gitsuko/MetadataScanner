# MetadataScanner
A simple **metadata scanner** written in C++ with both **CLI** and **Qt GUI**.

## Build Instructions

### CLI version
Compile and run:
```bash
g++ main.cpp -o main
./main
```

### Qt GUI version
Using CMake:
```bash
cmake .
make
./metadatascanner
```
